Que es Virtualización

La virtualización es una tecnología que permite crear múltiples entornos simulados o recursos dedicados a partir de un único sistema de hardware físico. Utiliza software para crear una capa de abstracción sobre el hardware del sistema, permitiendo dividir los elementos de hardware de un sistema (procesadores, memoria, almacenamiento, etc.) en varios sistemas virtuales, comúnmente llamados máquinas virtuales (VM). Cada VM ejecuta su propio sistema operativo y se comporta como un ordenador independiente. Esto permite una utilización más eficaz del hardware físico y una mayor rentabilidad de la inversión de hardware de una organización.

Que es Docker

Docker es un software de código abierto que permite desplegar aplicaciones dentro de contenedores virtuales. Estos contenedores permiten que las aplicaciones funcionen en diferentes entornos sin problemas. Docker comparte el sistema operativo del anfitrión, a diferencia de las máquinas virtuales que tienen un sistema operativo invitado. Esto permite un rendimiento rápido y una mayor eficiencia en el uso de la memoria. Docker facilita la creación, diseño, envío y control de versiones de las imágenes de los contenedores. Además, permite la modularidad, lo que significa que se puede actualizar o reparar una parte de la aplicación sin necesidad de deshabilitarla por completo.
